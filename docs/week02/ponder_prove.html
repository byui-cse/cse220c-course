Week 2 Ponder and Prove: First C++ Program

Overview

Task: Write a console-based calculator program that works with a series of integers.

Purpose: Learn to write C++ programs.

Instructions

You will create a console-based calculator program that will accept a series of integers entered by the user, then display the sum, average, min, and max. The program will first prompt the user for the number of integers they will be entering. Then prompt the user for each number, one at a time. The program will then calculate the sum, average, min, and max for the given numbers and display the result on the console.

Requirements

  • Store the numbers entered by the user in an array of integers.
  • Use a unique_ptr and new to declare an array of the required size.
  • In addition to main, you must write three function definitions for these three function prototypes: 
    				
int calc_sum(int count, const unique_ptr<int[]>& numbers);
int find_min(int count, const unique_ptr<int[]>& numbers);
int find_max(int count, const unique_ptr<int[]>& numbers);
  • Do not use any built-in sum, min, or max functions. You must write them yourself.
  • Use a standard C++ for loop to calculate the sum, min, and max in each function. You may use the value returned by the calc_sum function to calculate the average.

Once your program is working using the three functions, now calculate the sum, average, min, and max again, but this time use the built-in reduce function (accumulate) to calculate the sum, min, and max.

Now add to your program. This time, let's practice using a map function. Use the C++ transform function on the array of numbers to square each number, storing the results back in the array in place of the original numbers. Use a standard C++ for loop to display the resulting list of numbers.

Next, let's use a filter function (remove_if) to find all numbers in the array of squares that are less than 1000. Use a for_each loop to display these numbers on the console.

Finally, use the remove_if function again to find all odd numbers in our array of numbers less than 1000 and display them using a for_each loop.

Hints

  • Use a standard C++ for loop to prompt for each number and insert it into the array
  • You may need to explicitly convert a value from int to float when calculating the average
  • You will need to include the algorithm library for the transform, remove_if, and for_each functions
  • You will need to include the numeric library for the accumulate function
  • You will need to include the memory library for unique_ptr
