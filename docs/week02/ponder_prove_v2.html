CSE 220c: Week 02

Ponder and Prove

Week 2 Ponder and Prove: Using Classes

Overview

Task: Create a simple bank system to keep track of account balances.

Purpose: Learn how to create C++ classes and use overloaded operators.

Instructions

These instructions are designed to help you break your programming into small pieces to make it easier for you to write and debug. You only turn in the final program, after all of the parts have been written, but it will be easier if you make sure that each part is complete before proceeding to the next.

Part 1.

Create a simple class that represents a bank account. For this task, you will create a program that has a single account. In later tasks, you will add the ability to store multiple accounts.

The account class will have three data members: an account ID (integer), account name (string), and an account balance (float). All of these members should be private or protected.

The account class will have a constructor that takes no parameters. This constructor will initialize the data members to empty values.

The account class will have a copy constructor that copies all of the data from one instance to another instance.

The account class will have a destructor that takes no parameters and an empty body. We'll add more code here later.

The account class will have a method that prompts the user to enter the name and the initial balance. The ID of the class will be automatically set by your program with the next ID number available. The first ID number is 0, and subsequent accounts will get numerically increasing ID numbers.

The account class will have a method that will send the data members to the output stream passed in as a parameter. The stream should be passed by reference to the method, and should return the same stream reference that is passed to the method.

Your main function will create an object of the account class, ask the user to enter the account information, and call the output method to show the data members to the user. You can use the standard object cout to pass to the output method.

Example input and output (input is underlined):

Enter the name: George
Enter the balance: 100
Account ID: 0  Name: George  Balance: $100.00


Part 2.

Write an output operator << to enable writing the object's data members to an output stream. Since you should already have a public method that can be used for this purpose, you merely need to call it from your output operator.  The output operator should not be a member of the account class.

Modify your main function to use the output operator instead of calling the output method directly, to send the object to cout.

Example input and output (input is underlined):

Enter the name: George
Enter the balance: 100
Account ID: 0  Name: George  Balance: $100.00


Part 3.

Add a menu to your program to allow the user to manipulate and view the account information. Your menu should have two options: display the account information and quit the program.

Example input and output (input is underlined):

Account ID: 0 Name: George Balance: $100.00
