C++ and Python: Containers

Containers in Python and C++

We will now cover how Python containers compare to C++ containers.

Array

The Python language has several containers built into the language: lists, sets, tuples, dictionaries, and others. The C++ language has one built-in container: the array. The array is a fixed-size container where every element is of the same datatype. Each element of an array is accessible through an index. Note that if there are 10 elements in an array, then only index 0 through 9 are valid. If you attempt to access index -1 or 10, then you will get an unexpected result.

PythonC++
# Create a list of four elements.
data = [2, 4, 6, 8]

# Display the third element.
print("data[2] = ",data[2])

# Display the number of elements.
print("size:", len(data))

# Display all the elements.
for datum in data:
    print("\t", datum)

#include <iostream>
using namespace std;

/****************************
 * MAIN
 ****************************/
int main()
{
   // Create an array of four elements.
   int data[] = {2, 4, 6, 8};

   // Display the third element.
   cout << "data[2] = "
        << data[2] << endl;

   // Display the number of elements.
   cout << "size: "
        << sizeof(data)/sizeof(data[0])
        << endl;

   // Display all the elements.
   for (auto datum : data)
      cout << '\t'
           << datum << endl;
   return 0;
}

Vector

C++ has many containers which are not part of the language but included in the standard template library. The most popular is the vector, a data structure nearly identical to Python's list. The vector allows you to add elements to the back with the push_back() method, query the length with the size() method, and provides a variety of other functionalities.

PythonC++
# Create a list of four elements.
data = [2, 4, 6, 8]

# Add an element to the end.
data.append(10);

# Display the number of elements.
print("size:", len(data))

# Remove all elements.
data.clear()

// Vector is part of the "vector" library
#include <vector>

#include <iostream>
using namespace std;

/****************************
 * MAIN
 ****************************/
int main()
{
   // Create a vector of four integers.
   vector<int> data = {2, 4, 6, 8};

   // Add an element to the end.
   data.push_back(10);

   // Display the number of elements.
   cout << "size:" 
        << data.size() << endl;

   // Remove all elements.
   data.clear();
   return 0;
}

Note that declaring a vector is different than other data-types because of the somewhat "unique" vector<int> notation. This notation means the vector class is a template.

Iterator

The traditional way to visit every element of an array is to do so by index. This index variable is an integer.

PythonC++
# Create a list of four elements.
data = [2, 4, 6, 8]

# Visit each element by index
for index in range(0, len(data)):
    print(data[index])

#include <iostream>
#include <vector>
using namespace std;

/****************************
 * MAIN
 ****************************/
int main()
{
   // Create a vector of four integers.
   vector<int> data = {2, 4, 6, 8};

   // Visit each element by index.
   int index;
   for (index = 0; 
        index < data.size(); 
        index++)
      cout << data[index] << endl;
    
    return 0;
}

An iterator is similar to an index except it knows which container it is pointing to. To get a value from an index, you need the container. To get a value from an iterator, only the iterator itself is required. An iterator retrieves the value in the container with the dereference operator *.

C++
#include <iostream>
#include <vector>
using namespace std;

/****************************
 * MAIN
 ****************************/
int main()
{
   // Create a vector of four integers.
   vector<int> data = {2, 4, 6, 8};

   // Visit each element by iterator.
   for (auto it = data.begin(); it != data.end(); it++)
      cout << *it << endl;
    
   return 0;
}

In the above code, note that we used the auto keyword to allow C++ to determine the datatype of the variable. 

auto it = data.begin();

We could also use the full variable data-type. Note that the iterator is a sub-class of the vector class. The full name for the data-type is the following: 

vector<int>::iterator it = data.begin();

Most people prefer to use the auto keyword because it is much easier to use.

It is also possible to go through a collection (list or vector) backwards. To do this, we use a reverse iterator. The declaration for the vector's reverse iterator is vector<int>::reverse_iterator rit;. The last element of the collection is reached through rbegin() rather than begin(). The code to walk through four elements backwards is:

C++
#include <iostream>
#include <vector>
using namespace std;

/****************************
 * MAIN
 ****************************/
int main()
{
   // Create a vector of four integers.
   vector<int> data = {2, 4, 6, 8};

   // Visit each element by iterator backwards.
   for (auto rit = data.rbegin(); rit != data.rend(); rit++)
      cout << *rit << endl;
    
   return 0;
}

List

The list container is similar to the vector with one exception: it does not implement the [ subscript ] operator. This means the only way to access each of the elements is through an iterator. The list also facilitates efficient insertion and deletion from the middle (unlike the vector which is slow when inserting or deleting from the middle). Note that Python does not have a container similar to the list.

C++
#include <iostream>
#include <list>
using namespace std;

/****************************
 * MAIN
 ****************************/
int main()
{
   // Create a list of four integers.
   list<int> data = {2, 4, 6, 8};

   // Create an iterator pointing to the first element.
   auto it = data.begin();
   ++it;

   // Insert an element in the middle of the list.
   data.insert(it, 5);

   // Visit each element by iterator.
   for (it = data.begin(); it != data.end(); it++)
      cout << *it << endl;
    
   return 0;
}
