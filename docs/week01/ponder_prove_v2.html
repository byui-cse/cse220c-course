CSE 220C: Week 01

Ponder and Prove

Week 1: C++ Basics

Overview

Task: Create a simple bank system to keep track of account balances.

Purpose: Learn how to declare variables, handle input and output, and loops.

Instructions

These instructions are designed to help you break your programming into small pieces to make it easier for you to write and debug. You only turn in the final program, after all of the parts have been written, but it will be easier if you make sure that each part is complete before proceeding to the next.

Before attempting to complete the task, you should read through the following material. These reading links are provided to help you understand what you'll need to do as you complete the task. You don't necessarily need to read every word of each link, but you should browse through each link and spend time on concepts that are not familiar to you.

  • CSE 220C: Week 01 Reading
  • Optional Demo Video showing how to start a new project and write a simple C++ program.
  • Learn C++: Basics - Optional additional reading. (NOTE: you should ignore the instructions on installing OpenGL, since we will not be using that for this course.)
  • Learn C++: Computation - Optional additional reading.

You may need to consult additional documentation not listed here. You should ask for help in our class discussion when you get stuck.

Each week's programming task will build upon the previous week's task, so it's very important to complete each task in a timely fashion. Each week, you should create a new project for the week's task and copy any code over from the previous week's project. When you submit your program code, submit all of your files together in a single zip file.

Part 1.

Create a new C++ project in your IDE. If given the choice, it should be a console application.

In your main function, declare three variables. The first will be an identification number of type int, which should be set to 0. The second will be a string that represents the name of the bank account holder. The third will be a floating point number that represents the balance in the bank account.

Next, prompt the user to enter the name on the account and the initial account balance.

Run your program and it should look something like this (user input is underlined). Enter the name: George
Enter the balance: 120.50

Part 2.

Create a loop inside your main function. Use the loop to display a menu of options to the user. The menu should have two options. Option 0 is to quit the program and option 1 is to display the account information.

The menu should be presented to the user and allow the user to enter a choice. The program should not exit until the user chooses that option from the menu.

Add an option in your menu to display the account information. Use std::cout and the << operator to display this information.

Enter the name: George
Enter the balance: 120.50

Account Menu:
0. Quit Program
1. Display Account Information
Your choice: 1
Account ID: 0  Name: George  Balance: $120.50
Your choice: 0

Part 3.

Add an option in your menu to add a deposit to the account. Ask the user for the amount to deposit, and read in the amount. Add the new amount to the existing balance.

Part 4.

Add an option in your menu to make a withdrawal from the account. Ask the user for the amount to withdraw, and read in the amount. Subtract the new amount to the existing balance.
