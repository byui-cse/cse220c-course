CSE 220C: Week 01

Ponder and Prove

Week 1: C++ Basics

Overview

Task: Create a simple bank system to keep track of account balances.

Purpose: Learn how to declare variables, handle input and output, and loops.

Instructions

These instructions are designed to help you break your programming into small pieces to make it easier for you to write and debug. You only turn in the final program, after all of the parts have been written, but it will be easier if you make sure that each part is complete before proceeding to the next.

These links may help you:

  • CSE 220C: Week 01 Reading
  • CSE 230: C++ Basics
  • CSE 230: Computation

You may need to consult additional documentation not listed here. You should ask for help in our class discussion when you get stuck.

Part 1.

In your main function, declare three variables. The first is an identification number of type int, which should be set to 0. The second is a string that represents the name of the bank account holder. The third is a floating point number that represents the balance in the bank account. Prompt the user to enter the name on the account and the initial account balance.

Part 2.

Create a loop inside your main function. Inside the loop, present a menu of options to the user. For now, the only option will be to quit the program. The menu should be presented to the user and the user's choice should be input. Only when the user chooses to quit the program should the program exit.

Part 3.

Add an option in your menu to display the account information. Use std::cout and the << operator to display this information.

Part 4.

Add an option in your menu to add a deposit to the account. Ask the user for the amount to deposit, and read in the amount. Add the new amount to the existing balance.

Part 5.

Add an option in your menu to make a withdrawal from the account. Ask the user for the amount to withdraw, and read in the amount. Subtract the new amount to the existing balance.
