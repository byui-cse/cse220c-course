This section intends to help your understanding of C++ by comparing to Python directly. As many of you are familiar with Python, you can study the following to aid in translating basic concepts to C++. Try out the code as you go in your C++ IDE by creating a small test project and running the short examples.

Hello World

Python is called a "multi-paradigm programming language." This means it can be written in the imperative style where a statement can be executed without a function. It can be written in the procedural style where every statement is inside a function. It can also be written in the object-oriented style where every statement is in a method which is in a class. C++ is a procedural language. Even a simple one-line program such as "Hello World" needs to have at least one function. That function is called main().

Note that C++ does not have any built-in input/output (I/O) functionality. To print text on the screen, we need the iostream library. To indicate we want to use this library we need to include it with #include <iostream>. All the code in the iostream library utilizes the standard namespace. The full name of the output stream is std::cout. We can use the short cout version of the name with the using namespace std; line at the head of the program.

The program "Hello World" in Python and in C++ is the following:

PythonC++
print("Hello world")

#include <iostream>
using namespace std;

int main()
{
   cout << "Hello world\n";
   return 0;
}

main()

Many Python programmers prefer to put all their code in functions and have the first function called main(). This is not necessary, but is a common practice. In C++, all programs must contain a function called main().

PythonC++
def main():
    '''This function will send 
       a greeting to the user.'''

    # Display the actual greeting.
    print("Hello World")

main()

// This is where cout is defined
#include <iostream>  

// "cout" instead of "std::cout"
using namespace std; 

/************************************
 * This function will send a greeting
 * to the user.
 ************************************/
int main()
{
   // Display the actual greeting.
   cout << "Hello world\n";
   return 0;
}

Comments and Style

Comments serve the same purpose in C++ as they do in Python: to explain to the readers of the source code how the program is to work. Though they serve the same purpose, comments in C++ use a different syntax.

Note that all function declarations should be accompanied by a comment describing what the function does. In Python, we use a triple-quote doc string to comment a function. 

''' Docstring comments for Python function comment blocks '''

In C++, we use block comments, usually with extra *'s.

/* Block Comments for C++ function comment blocks */

We also put line comments in the code itself. In Python, line comments start with a pound sign. 

# Line comments start with a pound sign in Python.

In C++, line comments start with a double-slash.

// Line comments start with a double-slash in C++
PythonC++
def function():
    ''' Docstring comments are placed
        inside function definitions. '''
    pass

# Line comments must be 
# complete sentences.
function()

#include <iostream>  
using namespace std; 

/************************************
 * Function comment blocks go before
 * function definitions
 ************************************/
int main()
{
   // Line comments can be 
   // complete sentences.
   cout << "Hello world"
        << endl; /* Block comment */
   return 0;
}

The Python language has a style guide defined by PEP 8 -- Style Guide for Python Code. There is no unified style guide for C++; every organization seems to have their own standards for naming variables and indenting code. BYU—Idaho has a style guide as well: C++ Style for BYU—Idaho

Output

Textual output to the screen in Python is handled through the print() function. Textual output in C++ is handled through the cout object and the insertion operator <<.

PythonC++
# Put text on the screen.
print("Hello world")

# Put numbers on the screen.
print(3.14159)

# Mix numbers and text.
age = 24;
print("My age is", age)
print(f"My age is {age}")

#include <iostream>
using namespace std;

/*************************
 * Main
 *************************/
int main()
{
   // Put text on the screen.
   cout << "Hello world\n";

   // Put numbers on the screen.
   cout << 3.14159 << endl;

   // Mix numbers and text.
   int age = 24;
   cout << "My age is "
        << age 
        << endl;
   return 0;
}

See also: C++: std::cout


It is possible to configure cout to display floating point numbers. There are two parts:

  • fixed: Tells C++ not to use scientific notation
  • precision: Tells C++ to use a certain number of digits after the decimal point
PythonC++
# Puts $931.12 on the screen.
dollars = 931.12322
print(f"${dollars:.2f}")

# Puts 3.9 on the screen.
gpa = 3.871
print(f"{gpa:.1f}")

#include <iostream>
using namespace std;

/*************************
 * Main
 *************************/
int main()
{
   // Puts $931.12 on the screen.
   double dollars = 931.12322;
   cout.setf(ios::fixed);
   cout.precision(2);
   cout << "$"
        << dollars
        << endl;

   // Puts 3.9 on the screen.
   double gpa = 3.871;
   cout.precision(1);
   cout << gpa
        << endl;

   return 0;
}

See also: C++ Reference: std::ios_base::precision

Input

Input in Python is handled with the input() function. This always returns a string which the programmer may need to convert before use. Input in C++ is handled with the extraction operator >>. Note that a variable must be declared before it can be used with an extraction operator. We will learn more about variable declarations next week.

PythonC++
# Prompt for name.
name = input("Name? ")

# Prompt for age.
age = int(integer("Age? "))

// Use the iostream library for 
// cout and cin
#include <iostream>

// Use the string library for string
#include <string>

// Use the standard namespace so we can
// say "cout" rather than "std::cout".
using namespace std;

/*************************
 * Main
 *************************/
int main()
{
   // Prompt for name.
   string name;
   cout << "Name? ";
   cin  >> name;

   // Prompt for age.
   int age;
   cout << "Age? ";
   cin  >> age;

   return 0;
}

See also: C++ Reference: std::cin

It is also possible to prompt the use for an entire line of text. Use the getline() function for that.

PythonC++
# Prompt for full name.
name = input("Full Name: ")

# Display full name.
print("Full name is:", name)

#include <iostream> // for cout, cin
#include <string>   // for string
using namespace std;

/*************************
 * Main
 *************************/
int main()
{
   // Prompt for full name.
   string name;
   cout << "Full Name: ";
   getline(cin, name);

   // Display full name.
   cout << "Full name is: "
        << name
        << endl;

   return 0;
}

Functions

You can declare your own function in C++ just as you can in Python. The syntax is different of course. In C++, you must explicitly state the return type, even if no value is returned. You also enclose the body of the function in { curly braces }.

PythonC++
def my_simple_function():
    ''' No parameters, 
        no return value. '''
    print("Hello world")

def return_a_value():
    ''' Only a return value. '''
    return 42

def take_two_parameters(x, y):
    '' Takes two parameters. '''
    print("(x,y) = {x},{y}")

# Call the three functions
my_simple_function()
x = return_a_value()
take_two_parameters(4, 5)

#include <iostream>
using namespace std;

/*************************
 * No parameters, no return
 *************************/
void mySimpleFunction()
{
   cout << "Hello world\n";
}

/*************************
 * Only a return value
 *************************/
int returnAValue()
{
   return 42;
}

/*************************
 * Takes two parameters
 *************************/
void takeTwoParameters(int x, int y)
{
   cout << "(x,y) ="
        << x << ',' << y
        << endl;
}

/*************************
 * Main
 *************************/
int main()
{
   // Call the three functions
   mySimpleFunction();
   int x = returnAValue();
   takeTwoParameters(4, 5);

   return 0;
}

Notice how comment blocks are done differently in C++ than in Python. The most important difference is the comment which appears before the function declaration in C++, not inside the function as it does in Python.
